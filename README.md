📦 Mediloop

Medicine Verification & Safe Donation Platform

Built for MumbaiHacks 2025, Mediloop helps reduce medicine waste by connecting donors, NGOs, and organizations with verified medicine needs.
The platform verifies medicine details using intelligent scanning tools and ensures safe, transparent medicine reuse.


---

🌍 Problem

Every year, tons of safe & unused medicines are wasted because there is no structured system to verify, share, and donate them.
Meanwhile, NGOs and under-resourced communities struggle to access essential medicines.

Mediloop bridges this gap.


---

🚀 What Mediloop Does

✔ Allows donors to list unused, safe medicines
✔ Verifies medicine details using OCR + barcode scanning
✔ Matches donors with NGOs / needy recipients
✔ Encourages safe redistribution and reduces medical waste
✔ Ensures transparency & traceability in the donation process


---

🧠 Tech Stack

Frontend: React + TypeScript

AI / Scanning Tools:

Tesseract.js (OCR for text extraction)

ZXing (barcode decoding)

QuaggaJS (real-time scanning)


Other Tools:

Modern UI components

API integration for verification (optional)




---

🛠 How to Run Locally

npm install
npm run dev

Open in browser at:

http://localhost:3000


---

📸 Features in Detail

🔍 1. Medicine Verification

Extracts text using Tesseract.js OCR

Reads barcodes / QR codes using ZXing & Quagga

Matches medicine name, expiry date, batch ID



---

♻️ 2. Safe Donation Flow

Donors upload medicine details

System verifies validity (expiry, manufacturer info)

NGOs can browse verified donations

Reduces unnecessary disposal



---

🤝 3. Connecting Communities

Smart matching between donors & NGOs

Location-based suggestions

Status tracking for accepted donations



---

🌱 Mission

Mediloop’s goal is to eliminate avoidable medicine waste and ensure that safe unused medicine reaches people who actually need it — responsibly and securely.
